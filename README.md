## What is this

My website

## How to build

### Build all.

```
$ make all
```

### Build top page without blog

```
$ make index
```

### Build blog site

```
$ make blog
```

### Clean the generated product

```
$ make clean
```

## How to test

### Serve top page without blog

```
$ make test-index
```

### Serve blog site

```
$ make test-blog
```
